Crea y trabaja con un objeto Hashtable o HashMap, a partir de los
datos de tutor contenidos en fichero DatosTutor.

Puedes permitir al usuario que consulte datos de un tutor para lo
que se le pide introduzca (introducir por teclado) el id del mismo.

Para ello debes construir el hash con los datos del fichero y luego
en una 'repetitiva' aceptar consultas,por ejemplo hasta que desee
terminar opción ésta que debes ofrecer al usuario.

El hash puede tener como clave el 'id de tutor' y como valor un 
objeto Tutor:

    <Integer,Tutor>

Recordando que el hash será un conjunto de key/valor.
